Design Elementer

Elementerne i spillet er lavet i Adobe Illustrator og er alle tegnet i Vector format, og er derfor ikke opbygget af pixels som et normalt billede. Inden det endelig resultat, har vi skitseret figurerne for at finde den ønskede still. Mit endte med at ligne south park, hoved figuren det er baseret på mindre om en karakter der fra.

CSS Animation

I spillet falder elementerne ned fra himmel og når de klikkes på forsvinder de, typisk med en animation som for dem til at rotere eller blive mindre. Dette opnås i CSS ved hjælpe a @Keyframes og properties “animation”. Med dem kan vi beslutte hvordan de falder, hvor hurtigt, hvor mange gang og timing-funktionen.

Javascript

Javascript laders os manipulere med vores HTML DOM(html dokumentet, samt vores CSS file. I denne tog vi kun udgangs punkt i vores HTML DOM. Ved hjælp af “functions” kan vi udfører en række beslutninger, vi kan starte vores “functions” gennem en Event Listener method, til at holde øje med når vi klikker på en at vores HTML Element i spillet.
